线性插值与双线性插值

线性插值

在二维平面上已知两点(x1,y1)(x_1, y_1)(x1​,y1​)和(x2,y2)(x_2, y_2)(x2​,y2​), 想知道在[x1,x2][x_1, x_2][x1​,x2​]中的横坐标为xxx的点的纵坐标是多少?

如上图, 根据直线的斜率是一致的可以列出以下等式

y−y1x−x1=y2−y1x2−x1\frac{y-y_1}{x-x_1} = \frac{y_2-y_1}{x_2-x_1} x−x1​y−y1​​=x2​−x1​y2​−y1​​

转换得到

y=y2−y1x2−x1(x−x1)+y1y = \frac{y_2-y_1}{x_2-x_1}(x-x_1) + y_1 y=x2​−x1​y2​−y1​​(x−x1​)+y1​

转换成y1y_1y1​和y2y_2y2​的加权和的形式

y=x2−xx2−x1y1+x−x1x2−x1y2y = \frac{x_2-x}{x_2-x_1}y_1 + \frac{x-x_1}{x_2-x_1}y_2 y=x2​−x1​x2​−x​y1​+x2​−x1​x−x1​​y2​

双线性插值


双线性插值会通过两次线性插值得到, 首先通过(x0,y0)(x_{0}, y_{0})(x0​,y0​)和(x0,y1)(x_{0}, y_{1})(x0​,y1​)插值得到(x0,y)(x_0, y)(x0​,y), 然后通过(x1,y0)(x_{1}, y_{0})(x1​,y0​)和(x1,y1)(x_{1}, y_{1})(x1​,y1​)插值得到(x1,y)(x_1, y)(x1​,y), 最后通过(x0,y)(x_0, y)(x0​,y)和(x1,y)(x_1, y)(x1​,y)插值得到(x,y)(x, y)(x,y)
根据线性插值有如下公式

f(x0,y)=y1−yy1−y0f(x0,y0)+y−y0y1−y0f(x0,y1)f(x_0, y) = \frac{y_1-y}{y_1-y_0}f(x_0, y_0) + \frac{y-y_0}{y_1-y_0}f(x_0, y_1) f(x0​,y)=y1​−y0​y1​−y​f(x0​,y0​)+y1​−y0​y−y0​​f(x0​,y1​)

f(x1,y)=y1−yy1−y0f(x1,y0)+y−y0y1−y0f(x1,y1)f(x_1, y) = \frac{y_1-y}{y_1-y_0}f(x_1, y_0) + \frac{y-y_0}{y_1-y_0}f(x_1, y_1) f(x1​,y)=y1​−y0​y1​−y​f(x1​,y0​)+y1​−y0​y−y0​​f(x1​,y1​)

f(x,y)=x1−xx1−x0f(x0,y)+x−x0x1−x0f(x1,y)f(x, y) = \frac{x_1-x}{x_1-x_0}f(x_0, y) + \frac{x-x_0}{x_1-x_0}f(x_1, y) f(x,y)=x1​−x0​x1​−x​f(x0​,y)+x1​−x0​x−x0​​f(x1​,y)

Pytorch上的实现

可以参考github上面的这个issue
可以将一个像素看作是一个1*1的正方形

有两种对齐方式, 第一种是角对齐方式, 如上图1所示, 原图的角点和目标图像的角点是对齐的
第二种是边对齐方式, 目标图像的四条边和原图像的四条边对齐

有以下几个注意点:

  • 在边对齐的情况下, 可能会出现(x,y)(x, y)(x,y)在原图范围外的情况, 这个时候需要将其限制到原图内
  • 在角对齐的情况下, 对宽来说要将w−1w-1w−1放大到ori_w−1ori\_w-1ori_w−1, 对高同理
  • 在边对齐的情况下, 给定两个对应的像素点, 他们到左上角的角点的边长是成比例的, 因此需要加上0.5
  • 还有种情况是点在最右边或者最下边, 这种情况下会有点在原图外, 需要将点移到原图内
