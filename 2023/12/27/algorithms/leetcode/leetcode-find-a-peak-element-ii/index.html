思路

这个真有点想不到, 还是直接看证明吧, 目前应该是实力还不够

考虑第i1,i2i_1, i_2i1​,i2​行, i1<i2i_1 < i_2i1​<i2​, 第i1i_1i1​行的最大值mat[i1][ji1]>mat[i1−1][ji1]mat[i_1][j_{i_1}] > mat[i_1-1][j_{i_1}]mat[i1​][ji1​​]>mat[i1​−1][ji1​​]且第i2i_2i2​行的最大值mat[i2][ji2]>mat[i2+1][ji2]mat[i_2][j_{i_2}] > mat[i_2+1][j_{i_2}]mat[i2​][ji2​​]>mat[i2​+1][ji2​​]. 假设在i1∼i2i_1 \sim i_2i1​∼i2​行之间没有峰值, 那么有如下成立
mat[i1][ji1]<mat[i1+1][ji1]mat[i_1][j_{i_1}] < mat[i_1+1][j_{i_1}]mat[i1​][ji1​​]<mat[i1​+1][ji1​​] (任意两个格子不相同)
那么对于对于第i1+1i_1+1i1​+1行的最大值满足
mat[i1+1][ji1+1]≥mat[i1+1][ji1]>mat[i1][ji1]≥mat[i1][ji1+1]mat[i_1+1][j_{i_1+1}] \geq mat[i_1+1][j_{i_1}] > mat[i_1][j_{i_1}] \geq mat[i_1][j_{i_1+1}]mat[i1​+1][ji1​+1​]≥mat[i1​+1][ji1​​]>mat[i1​][ji1​​]≥mat[i1​][ji1​+1​]
也就是说下一行的最大值也必须满足大于上一行的对应值
那么根据数学归纳法可以一直证明到i2i_2i2​行的最大值也是大于它上方的数, 那么第i2i_2i2​行一定存在峰值点, 就和假设矛盾了

因此得出如下结论
如果第i1,i2i_1, i_2i1​,i2​行, i1<i2i_1 < i_2i1​<i2​, 第i1i_1i1​行的最大值mat[i1][ji1]>mat[i1−1][ji1]mat[i_1][j_{i_1}] > mat[i_1-1][j_{i_1}]mat[i1​][ji1​​]>mat[i1​−1][ji1​​]且第i2i_2i2​行的最大值mat[i2][ji2]>mat[i2+1][ji2]mat[i_2][j_{i_2}] > mat[i_2+1][j_{i_2}]mat[i2​][ji2​​]>mat[i2​+1][ji2​​], 那么在第i1∼i2i_1 \sim i_2i1​∼i2​行之间一定存在峰值.

初始情况下i1=0,i2=m−1i_1 = 0, i_2 = m-1i1​=0,i2​=m−1
