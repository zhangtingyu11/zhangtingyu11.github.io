DID-M3D:Decoupling Instance Depth for Monocular 3D Object Detection

发表于2024-01-03|更新于2024-01-03|论文阅读3D目标检测纯相机3D目标检测
|字数总计:1.6k|阅读时长:6分钟|阅读量:

主要思路

将深度分为visual depth和attribute depth
对RoI中的每个网格都预测深度, 再将深度进行整合

Introduction


对于上图右边的物体, 物体的深度可以分解成到车尾的深度和车身的一半长度
对于上图左边的物体, 深度还受到车辆的长宽以及航向角的影响

本文提出将物体的深度进行分解, 分解成instance visual surface depth(visual depth)和instance attribute depth(attribute depth)
visual depth指的是到相机的绝对深度, attribute depth指的是从表面到3D中心点的偏差

visual depth依赖物体的外观和在图像中的位置, 这是affine-sensitive(对仿射变换敏感)
attribute depth依赖物体的内在属性(例如尺寸和航向), 主要关注RoI内部的特征, 这是仿射不变的

这两个深度是独立的, 因此可以使用两个不同的检测头来生成不同的深度
具体来说, 对于一个image patch, 将其划分成m×nm \times nm×n的网格, 每个网格都包含一个物体的小区域, 将会包含一个visual depth和对应的attribute depth
每个物体会生成m×nm\times nm×n个深度预测结果, 通过深度和其对应的不确定性来获取最终的深度和置信度

因为将深度进行了解耦, 可以进行仿射变换相关的数据增强

主要贡献

  • 将深度进行分解, 分解成visual depth和attribute depth
  • 自适应地整合不同类型的深度到最终的深度和对应的3D置信度中
  • 通过分解深度, 缓解了不能使用仿射变换的数据增强方法的限制
  • SOTA

Overview and Framework

Preliminaries

3D包围框由3D中心点位置(x,y,z)(x, y, z)(x,y,z), 尺寸(h,w,l)(h, w, l)(h,w,l)和航向θ\thetaθ组成

首先输入是一张图片I∈RH×W×3\mathbf{I} \in \R^{H\times W \times 3}I∈RH×W×3
编码特征后, 得到特征F∈RH4×W4×C\mathbf{F} \in \R^{\frac{H}{4} \times \frac{W}{4} \times C}F∈R4H​×4W​×C
然后将特征图送入2D检测头中, 得到三个输出:

  • 2D热力图H∈RH4×W4×B\mathbf{H} \in \R^{\frac{H}{4} \times \frac{W}{4} \times B}H∈R4H​×4W​×B, 其中BBB是类别数
  • 2D offset O2d∈RH4×W4×2\mathbf{O}_{2d} \in \R^{\frac{H}{4} \times \frac{W}{4} \times 2}O2d​∈R4H​×4W​×2
  • 2D 尺寸 S2d∈RH4×W4×2\mathbf{S}_{2d} \in \R^{\frac{H}{4} \times \frac{W}{4} \times 2}S2d​∈R4H​×4W​×2

给定2D包围框的估计, 可以通过RoI Align得到每个物体的的RoI特征
物体的RoI特征为Fobj∈Rn×7×7×C\mathbf{F}_{obj} \in \R^{n\times 7 \times 7 \times C}Fobj​∈Rn×7×7×C, 其中nnn表示RoI的个数

最后将RoI的特征送入3D检测头中生成3D参数:

  • 3D包围框的尺寸S3d\mathbf{S}_{3d}S3d​
  • 3D中心点投影值的offset O3d∈Rn×2\mathbf{O}_{3d} \in \R^{n \times 2}O3d​∈Rn×2
  • 航向角Θ∈Rn×k×2\Theta\in\mathbb{R}^{n\times k\times2}Θ∈Rn×k×2
  • visual depth Dvis∈Rn×7×7\mathbf{D}_{vis} \in \R^{n\times 7 \times 7}Dvis​∈Rn×7×7
  • visual depth uncertainty Uvis∈Rn×7×7\mathbf{U}_{vis} \in \R^{n\times 7 \times 7}Uvis​∈Rn×7×7
  • attribute depth Datt∈Rn×7×7\mathbf{D}_{att} \in \R^{n\times 7 \times 7}Datt​∈Rn×7×7
  • attribute depth uncertainty Uatt∈Rn×7×7\mathbf{U}_{att} \in \R^{n\times 7 \times 7}Uatt​∈Rn×7×7

Decoupled Instance Depth

将每个RoI分成7×77 \times 77×7个网格, 给每个网格一个visual depth和一个attribute depth

Visual Depth

visual depth表示物体表面的物理深度
对于每个网格, 定义visual depth是这个网格中的像素深度的平均值

在单目相机中, visual depth很大程度上和物体的2D包围框大小(远的物体看上去小)以及在图像上的位置(更小的vvv坐标, 深度越大)有关
当对图像进行仿射变换时, visual depth也要相对应的做变换

Attribute Depth

attribute depth表示从物体表面到物体的3D中心的深度差
attribute depth和物体的尺寸以及航向有关, 它与仿射变换无关, 因为仿射变换不会改变物体内在的属性

Data Augmentation


上图中对图像进行了随机裁剪和缩放. 那么3D中心点的投影也进行了同样的仿射变化. 根据公式d=f⋅h3dh2dd=\frac{f\cdot h_{3d}}{h_{2d}}d=h2d​f⋅h3d​​, 物体的visual depth也进行了和物体的y轴缩放比例一致的缩放

Depth Uncertainty and Aggregation

只有当物体有低visual depth uncertainty和低attribute depth uncertainty, instance depth才能有比较高的置信度


假设visual depth和attribute depth都满足Laplace分布(参考GUPNet)
使用以下公式获取置信度Pins(patch)=exp(−U~ins(patch))\mathbf{P}_{ins(patch)}=exp(-\mathbf{\tilde{U}}_{ins(patch)})Pins(patch)​=exp(−U~ins(patch)​)

也就是对7×77 \times 77×7的网格, 对每个网格计算visual depth和attribute depth的不确定性, 然后用公式u~ins=uvis2+uatt2\tilde{u}_{ins}=\sqrt{u_{vis}^{2}+u_{att}^{2}}u~ins​=uvis2​+uatt2​​来计算instance depth的不确定性, 再将不确定性转化成7×77 \times 77×7的置信度

对于深度而言, 也对7×77 \times 77×7的网格计算每个网格的深度, 然后使用公式d~ins=dvis+d~att\tilde{d}_{ins}=d_{vis}+\tilde{d}_{att}d~ins​=dvis​+d~att​来计算instance depth

最后的instance depth就是7×77 \times 77×7网格的加权平均

dins=∑D~ins(patch)iPins(patch)i∑Pins(patch)id_{ins}=\sum\frac{\mathbf{\tilde{D}}_{ins(patch)_i}\mathbf{P}_{ins(patch)_i}}{\sum\mathbf{P}_{ins(patch)_i}} dins​=∑∑Pins(patch)i​​D~ins(patch)i​​Pins(patch)i​​​

最后的depth confidence通过以下公式得出

pins=∑(Pins(patch)i∑Pins(patch)iPins(patch)i)p_{ins}=\sum(\frac{\mathbf{P}_{ins(patch)_i}}{\sum\mathbf{P}_{ins(patch)_i}}\mathbf{P}_{ins(patch)_i}) pins​=∑(∑Pins(patch)i​​Pins(patch)i​​​Pins(patch)i​​)

物体最后的置信度是p=p2d⋅pinsp=p_{2d}\cdot p_{ins}p=p2d​⋅pins​, 其中p2dp_{2d}p2d​是2D目标检测置信度
