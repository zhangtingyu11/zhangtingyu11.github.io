Geometry Uncertainty Projection Network for Monocular 3D Object Detection

主要思路

Introduction

现有的方法经常会先估计2D和3D包围框的高度, 然后通过公式h3d⋅f/h2dh_{3d}\cdot f/h_{2d}h3d​⋅f/h2d​来计算深度, 其中fff是相机的焦距

但是, 高度的预测误差是不可避免的. 2D高度的估计相对来说更加准确, 因此本文更关注3D高度估计的误差导致的深度误差, 由下图可以发现, 高度相差0.1m可能会导致深度相差4m.

在本文, 提出了一个Geometry Uncertainty Projection Network, 其中包含一个Geometry Uncertainty Projection模块和一个Hierarchical Task Learning策略来解决这些问题

首先第一个问题是预测的可靠性的问题, 高度预测上的小误差会导致深度预测的巨大误差. 为了解决这个问题, 提出预测深度的分布, 而不是单一预测一个离散的值. 深度的分布可以通过高度的分布得到

在预测阶段, 不确定性可以转化为置信度来表示深度预测的质量

另一个问题是模型训练的不可靠性. 在训练的初期, 2D/3D高度充满了噪音, 这些误差会导致错误的深度估计, 这样整个模型的训练方向就会改变

因此提出了Hierarchical Task Learning, 用于确保每个任务只会在它的前置任务训练好的情况下才会训练(例如深度预测需要等到3D高度训练好才会预测)

主要贡献

  • 提出了一个Geometry Uncertainty Projection(GUP)模块, 既包含了数学的先验, 又包含了不确定性的模拟. 减少了模型中误差的放大带来的影响
  • 任务之间存在依赖关系, 提出了Hierarchical Task Learning策略来提高训练的效率
  • SOTA

Geometry Uncertainty Projection Network

网络架构图如下

首先使用2D backbone来处理图像, 生成2D包围框, 然后计算基础的3D包围框的信息(角度, 尺寸, 3D投影的中心点)
接着使用GUP模块来预测深度的分布, 深度的分布提供了精确的深度值和对应的不确定性
预测出来的不确定性会映射到3D检测置信度用于预测阶段

2D detection

2D检测器是基于CenterNet的, 包含一个backbone网络和3个2D检测头(用于计算位置, 尺寸, 置信度).
heatmap会生成W×H×CW\times H \times CW×H×C尺寸的热力图来表示物体的位置和置信度
2D offset会计算和粗粒度的物体中心点的偏差(δ2du,δ2dv)(\delta_{2d}^u, \delta_{2d}^v)(δ2du​,δ2dv​)
2D尺寸会预测2D包围框的尺寸(w2d,h2d)(w_{2d}, h_{2d})(w2d​,h2d​)
他们的损失函数记为Lheatmap,Loffset2d,Lsize2d\mathcal{L}_{heatmap},\mathcal{L}_{offset2d}, \mathcal{L}_{size2d}Lheatmap​,Loffset2d​,Lsize2d​

RoI feature representation

为了让模型关注物体, 使用RoIAlign来裁剪和resize RoI的特征
但是这些特征缺少位置和尺寸的线索. 因此计算归一化的坐标图, 然后将其于每个RoI的特征图进行拼接

Basic 3D detection Heads

给定提取后的RoI特征, 在这些特征的基础上构建多个检测头, 用来预测一些基础的3D包围框信息(计算3D中心点的投影点的offset, 预测相对旋转角, 3D尺寸)
这些预测值的loss为Loffset3d,Langle,Lsize3d\mathcal{L}_{offset3d},\mathcal{L}_{angle}, \mathcal{L}_{size3d}Loffset3d​,Langle​,Lsize3d​
其中Lsize3d\mathcal{L}_{size3d}Lsize3d​包含三个尺寸的loss

Geometry Uncertainty Projection

首先假设3D高度的预测满足拉普拉斯分布La(μh,λh)La(\mu_h, \lambda_h)La(μh​,λh​)

在训练时, 不是直接预测λh\lambda_hλh​, 而是预测均值μh\mu_hμh​和标准差σh=2λ\sigma_h = \sqrt{2}\lambdaσh​=2​λ (这里的预测不是有监督学习, 而是无监督学习)

损失函数如下

Lh3d=2σh∣μh−h3dgt∣+log(σh)\mathcal{L}_{h3d}=\frac{\sqrt{2}}{\sigma_h}|\mu_h-h_{3d}^{gt}|+log(\sigma_h) Lh3d​=σh​2​​∣μh​−h3dgt​∣+log(σh​)

其中log(σh)log(\sigma_h)log(σh​)表示的是不确定性, 当方差(不确定性)越大时, 前面一项的权重就会越小, 当方差(不确定性)越小时, 前面一项的权重就会越大

根据下面的公式,

dp=f⋅h3dh2d=f⋅(λh⋅X+μh)h2d=f⋅λhh2d⋅X+f⋅μhh2d,\begin{aligned} d_{p}& =\frac{f\cdot h_{3d}}{h_{2d}}=\frac{f\cdot(\lambda_{h}\cdot X+\mu_{h})}{h_{2d}} \\ &=\frac{f\cdot\lambda_h}{h_{2d}}\cdot X+\frac{f\cdot\mu_h}{h_{2d}}, \end{aligned} dp​​=h2d​f⋅h3d​​=h2d​f⋅(λh​⋅X+μh​)​=h2d​f⋅λh​​⋅X+h2d​f⋅μh​​,​

其中X∼La(0,1)X \sim La(0, 1)X∼La(0,1), 可以发现, 深度的分布服从La(f⋅μhh2d,f⋅λhh2d)La(\frac{f\cdot\mu_h}{h_{2d}}, \frac{f\cdot\lambda_h}{h_{2d}})La(h2d​f⋅μh​​,h2d​f⋅λh​​)

通过高度公式得到深度后, 和标签的深度可能还存在误差, 因此要增加一个bias, 假设bias也服从拉普拉斯分布, 并且和dpd_pdp​独立
最终的深度, 均值和方差如下

d=La(μp,σp)+La(μb,σb),μd=μp+μb,σd=(σp)2+(σb)2.\begin{gathered}d=La(\mu_p,\sigma_p)+La(\mu_b,\sigma_b),\\\mu_d=\mu_p+\mu_b,\quad\sigma_d=\sqrt{(\sigma_p)^2+(\sigma_b)^2}.\end{gathered} d=La(μp​,σp​)+La(μb​,σb​),μd​=μp​+μb​,σd​=(σp​)2+(σb​)2​.​

最后的不确定性σd\sigma_dσd​记为Geometry based Uncertainty(GeU)
最终的深度也要计算不确定性, 损失函数如下

Ldepth=2σd∣μd−dgt∣+log(σd)\mathcal{L}_{depth}=\frac{\sqrt{2}}{\sigma_d}|\mu_d-d^{gt}|+log(\sigma_d) Ldepth​=σd​2​​∣μd​−dgt∣+log(σd​)

在预测的过程中, 将深度的不确定性σd\sigma_dσd​映射到0~1, 作为一个Uncertainty-Confidence(UnC)

pdepth=exp(−σd)p_{depth}=exp(-\sigma_d) pdepth​=exp(−σd​)

最后的置信度是2D置信度和UnC的乘积

p3d=p3d∣2d⋅p2d=pdepth⋅p2dp_{3d}=p_{3d|2d}\cdot p_{2d}=p_{depth}\cdot p_{2d} p3d​=p3d∣2d​⋅p2d​=pdepth​⋅p2d​
