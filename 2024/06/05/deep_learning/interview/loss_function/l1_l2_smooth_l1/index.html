MSE损失函数

介绍

MSE又称为Mean Squared Error损失函数, 平方损失函数
数学表达式为

MSE=1n∑i=1n(yi−y^i)2MSE=\frac1n\sum_{i=1}^n(y_i-\hat{y}_i)^2 MSE=n1​i=1∑n​(yi​−y^​i​)2

其中nnn是样本数量, yiy_iyi​是第iii个样本的真实标签, yi^\hat{y_i}yi​^​是第iii个样本的预测值

特点
会放大离群值的损失

pytorch接口

输入参数
size_average: 弃用, 请使用reduction
reduce: 弃用, 请使用reduction
reduction: 可以选择三个. ‘none’, ‘mean’, ‘sum’. 如果是’none’, 就不会进行reduction, 如果是’mean’, 就会除输出元素的个数, 如果是’sum’, 就会将输出元素相加

MAE损失函数

介绍

其实就是L1损失函数, 数学表达式如下

MAE=1n∑i=1n∣yi−y^i∣MAE=\frac1n\sum_{i=1}^n\mid y_i-\hat{y}_i\mid MAE=n1​i=1∑n​∣yi​−y^​i​∣

其中nnn是样本数量, yiy_iyi​是第iii个样本的真实标签, yi^\hat{y_i}yi​^​是第iii个样本的预测值

pytorch接口

输入参数
size_average: 弃用, 请使用reduction
reduce: 弃用, 请使用reduction
reduction: 可以选择三个. ‘none’, ‘mean’, ‘sum’. 如果是’none’, 就不会进行reduction, 如果是’mean’, 就会除输出元素的个数, 如果是’sum’, 就会将输出元素相加

SmoothL1 loss

介绍

SmoothL1 Loss的定义如下

LsmoothL1(x,y)={0.5(x−y)2/β,if∣x−y∣<β∣x−y∣−0.5×β,otherwiseL_{smoothL1}(x,y)=\begin{cases}0.5(x-y)^2/\beta,&\text{if}|x-y|<\beta\\|x-y|-0.5\times\beta,&\text{otherwise}\end{cases} LsmoothL1​(x,y)={0.5(x−y)2/β,∣x−y∣−0.5×β,​if∣x−y∣<βotherwise​

pytorch接口

输入参数
size_average: 弃用, 请使用reduction
reduce: 弃用, 请使用reduction
reduction: 可以选择三个. ‘none’, ‘mean’, ‘sum’. 如果是’none’, 就不会进行reduction, 如果是’mean’, 就会除输出元素的个数, 如果是’sum’, 就会将输出元素相加
beta: 指定从L1到L2 loss转变的阈值

三个函数的区别

记z=x−yz = x-yz=x−y
MAE对zzz的导数为

∂LL1(x,y)∂z={1,ifz≥0−1,otherwise\frac{\partial L_{L1}(x,y)}{\partial z}=\begin{cases}1,&\text{if}z\geq0\\-1,&\text{otherwise}\end{cases} ∂z∂LL1​(x,y)​={1,−1,​ifz≥0otherwise​

MSE对zzz的导数为

∂LL2(x,y)∂z=z\frac{\partial L_{L2}(x,y)}{\partial z}=z ∂z∂LL2​(x,y)​=z

SmoothL1对zzz的导数为

∂LsmoothL1(x,y)∂z={z/β,if∣x−y∣<1±1,otherwise\frac{\partial L_{smoothL1}(x,y)}{\partial z}=\begin{cases}z/\beta,&\text{if}|x-y|<1\\\pm1,&\text{otherwise}\end{cases} ∂z∂LsmoothL1​(x,y)​={z/β,±1,​if∣x−y∣<1otherwise​

由损失函数对zzz的导数可知

  • L1损失函数对zzz的导数为常数, 在训练后期, 误差值zzz很小时, 如果学习率bubian,损失函数会在稳定值附近波动, 很难收敛到更高的精度
  • L2损失函数对zzz的导数等于zzz, 在训练初期, zzz的值比较大的时候, 导数也比较大, 导致训练初期不稳定

SmoothL1的缺点
在计算目标检测的时候, 中心点坐标, 长宽都是单独计算loss, 但是他们其实是有关联的, SmoothL1没有考虑这种关联, 因此后面就出现了IoU Loss
