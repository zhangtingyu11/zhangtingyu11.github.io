目标检测任务评价指标

发表于2024-06-06|更新于2024-06-26|深度学习评价指标
|字数总计:950|阅读时长:3分钟|阅读量:

TP, FP, TN, FN

TP(针对检测框): 置信度大于阈值, 且IoU >= 阈值的检测框的数量
FP(针对检测框): 置信度大于阈值, 且IoU < 阈值的检测框的数量. 如果有多个预测框匹配到GT框, 且IoU都大于等于阈值, 那么就选择IoU最高的检测框作为TP, 其他的检测框作为FP
TN(针对检测框): 置信度小于阈值的检测框的数量
FN(针对gt框): 漏检的gt框, 等于gt框的个数-TP

Precision, Recall

Precision=TPTP+FPPrecision=\frac{TP}{TP+FP} Precision=TP+FPTP​

Recall=TPTP+FNRecall=\frac{TP}{TP+FN} Recall=TP+FNTP​

PR曲线

PR曲线的横坐标表示召回率, 纵坐标表示精确度
曲线上的每一个点都对应模型在特定阈值下的精确率和召回率
P-R曲线越接近于右上角越好

流程示例
假设总共有10个样本, 其中正样本4个, 负样本6个
模型预测出这10个样本属于正样本的概率是[0.1, 0.2, 0.3, 0.4, 0.44, 0.5, 0.6, 0.14, 0.43, 0.8]

  1. 将概率从大到小排序
  2. 按照排序的概率作为阈值, 大于等于阈值的预测结果为正样本, 小于阈值的预测结果为负样本, 在此基础上计算Precision和Recall(如果有概率值相同的计算一次即可)
  3. 将计算出来的点画在图像上, 将点连起来就是P-R曲线

AP, mAP

对于单个类别的目标检测, AP是P-R曲线下的面积, 用于度量模型在单个类别上的性能
对于多类别目标检测, mAP是所有类别AP的平均值

在计算PR曲线的面积之前, 需要修改每个点的位置, 将每个点的Precision值都改为该点本身及右边点的Precision的最大值

如何计算AP, mAP

首先计算AP
按照预测框的置信度从大到小进行排序, 会得到下面的表格的左半边, 每个预测框都要判断预测的是否正确, 如果这个包围框和GT框的IoU大于阈值且没有其他包围框和这个GT框的IoU比它更大, 那么这个包围框就是预测正确的, 否则就是预测错误的. 预测正确的记入TP里面, 预测错误的记入FP里面

这里GT框的个数是5
以第一个包围框的置信度为阈值, TP = 1, FN = 4, FP = 0, 因此Precision = 1.0, Recall=0.2
以第二个包围框的置信度为阈值, TP = 2, FN = 3, FP = 0, 因此Precision = 1.0, Recall=0.4
以第三个包围框的置信度为阈值, TP = 2, FN = 3, FP = 1, 因此Precision = 0.67, Recall=0.4
以第四个包围框的置信度为阈值, TP = 2, FN = 3, FP = 2, 因此Precision = 0.5, Recall=0.4
以第五个包围框的置信度为阈值, TP = 2, FN = 3, FP = 3, 因此Precision = 0.4, Recall=0.4
以第六个包围框的置信度为阈值, TP = 3, FN = 2, FP = 3, 因此Precision = 0.5, Recall=0.6
以第七个包围框的置信度为阈值, TP = 4, FN = 1, FP = 3, 因此Precision = 0.57, Recall=0.8
以第八个包围框的置信度为阈值, TP = 4, FN = 1, FP = 4, 因此Precision = 0.5, Recall=0.8
以第九个包围框的置信度为阈值, TP = 4, FN = 1, FP = 5, 因此Precision = 0.44, Recall=0.8
以第十个包围框的置信度为阈值, TP = 5, FN = 0, FP = 5, 因此Precision = 0.5, Recall=1.0
